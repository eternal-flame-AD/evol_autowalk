## 教程

1. 安装Pillow库
	 > git install Pillow                             （Windows）

2. 连接手机或模拟器打开usb调试 

3. 进入城市漫步

4. 运行evol_auto.py（MacOS需自行配置adb和Pillow后运行evol_auto_macos.py），如需要网络ADB 输入y之后输入地址（experimental)否则输入n

5. Ctrl+C中断或程序自动判断已刷完
